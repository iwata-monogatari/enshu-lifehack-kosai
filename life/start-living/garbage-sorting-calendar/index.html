引っ越し先で、ごみを間違えずに出せるようにする。

ごみ分別収集カレンダーは、引っ越し先の地区で「何を」「いつ」出せばいいかをまとめて確認できる資料です。湖西市はA地区・B地区で内容が分かれているので、まず自分の地区版を手に入れましょう。

カレンダーは年度ごとに更新されます。前の住居で使っていたカレンダーをそのまま使うと収集日を間違えることがあるため、転入後は必ず最新版を確認してください。

先に知っておきたいこと

📆 A地区・B地区でカレンダーが違う

A地区は新所・大森・岡崎など、B地区は表鷲津・鷲津などが対象です。自分の住所がどちらか分からない場合は廃棄物対策課に確認してください。

🌐 多言語版もある

日本語のほか、ポルトガル語、スペイン語、英語、中国語、インドネシア語、ベトナム語のカレンダーが用意されています。

📲 アプリでも同じ内容を確認できる

紙のカレンダーを紛失した場合や、通知機能を使いたい場合はごみ分別アプリが便利です。

🏭 自己搬入という選択肢もある

収集日を待てない場合は、環境センターへ自分でごみを持ち込む方法もあります（事前確認・料金あり）。

不安を先につぶすQ&A

カレンダーはどこでもらえますか。

転入時の窓口で配布されるほか、廃棄物対策課や公式サイトからも入手できます。ごみ分別収集カレンダーのページを確認してください。

自分の地区がA地区かB地区か分からない場合はどうすればいいですか。

ごみ出し検索のページで住所を入力すれば確認できます。廃棄物対策課（053-577-1280）への電話確認も可能です。

祝日の場合、収集は休みになりますか。

地区・時期によって取り扱いが異なる場合があります。カレンダーに祝日の収集有無が明記されているので、そちらを確認してください。

年末年始のごみ出しはどうなりますか。

年末年始は通常と異なる収集スケジュールになります。年末年始のごみ・資源物の出し方についてのページを確認してください。

旧新居町エリアと湖西地区でカレンダーが違いますか。

令和6年4月からごみ出しルールが統一され、以前は分かれていた湖西地区（A・B）と新居地区の収集カレンダーが1つのパターンにまとめられています。

あわせて確認したい暮らし始めたのリンク

布団・家具・大きなごみを、自分で持ち込むか、取りに来てもらうか決める。提出先に出す証明書を、種類を間違えず最短で受け取る。近くの窓口に行ってから「ここでは扱っていません」を防ぐ。犬の登録・鑑札の手続きを、出直しなしで一度に済ませる。毎年の狂犬病予防注射を、忘れずに受けて注射済票までそろえる。引っ越し直後でも、明日のごみを正しく出せるようにする。🧭 暮らし始めた引っ越してきた後の手続きを、抜けなく順番どおりに片づける。
